# 편의성 개선 사항

**날짜**: 2026-06-04

## 개요

Sacred Pixel Dungeon 웹 빌드에 적용된 종합적인 편의성 개선 사항 모음입니다. 키보드 접근성, 데이터 동기화, 로딩 최적화, 다국어 폰트 지원, 다양한 UI/UX 개선이 포함됩니다.

---

## 변경 사항

### 1. 키보드 접근성

마우스 없이 키보드만으로 게임의 주요 UI를 조작할 수 있습니다.

#### 1.1 글로벌 키 바인딩

| 키 | 액션 |
|----|------|
| W / ↑ / Numpad 8 | 위로 이동 (N) |
| X / ↓ / Numpad 2 | 아래로 이동 (S) |
| A / ← / Numpad 4 | 왼쪽 이동 (W) |
| D / → / Numpad 6 | 오른쪽 이동 (E) |
| S | 대기 / 아이템 줍기 |
| Enter / Numpad Enter | 선택 / 확인 |
| Escape / Backspace | 뒤로가기 / 취소 |
| Tab / Space / Numpad 0 | 다음 항목 순환 |

#### 1.2 키보드 네비게이션 지원 화면

| 화면 | 기능 |
|------|----------|
| TitleScene | 방향키로 버튼 간 이동, Enter로 선택 |
| HeroSelectScene | 좌/우로 영웅 선택, Enter로 게임 시작 |
| RankingsScene | 방향키로 기록 탐색, Enter로 상세 보기 |
| AmuletScene | 상/하로 Exit/Stay 선택 |
| SettingsScene | 방향키로 옵션 탐색, Enter로 선택/조정 |
| DataScene | 방향키로 버튼 탐색, Enter로 선택 |

#### 1.3 창 포커스 시스템

- **WndOptions**: 모든 옵션 버튼 키보드 탐색
- **WndBag**: 인벤토리 그리드 탐색
- **WndChooseSubclass/WndChooseAbility**: 서브클래스/갑옷 능력 선택
- **WndChallenges**: 도전과제 CheckBox 및 Info 버튼 탐색
- **WndJournal/JournalScene**: 탭 전환 + 컨텐츠 모드 탐색

#### 1.4 포커스 시각적 피드백

| 상태 | 표시 |
|------|----------|
| 버튼 포커스 | 배경/아이콘 밝기 130% 증가 |
| 선택됨 | 노란색 텍스트 (TITLE_COLOR) |

#### 1.5 TalentsPane 키보드 지원

- 2D 그리드 탐색 (티어 x 특성)
- 상/하: 티어 간 이동
- 좌/우: 같은 티어 내 특성 간 이동
- Enter: 특성 상세 정보/업그레이드

#### 1.6 OptionSlider 키보드 조정

- 포커스 상태에서 좌/우 방향키로 슬라이더 값 조정
- WndSettings의 모든 슬라이더에 적용

#### 1.7 카메라 이동 키보드 컨트롤

| 액션 | 기본 키 |
|------|---------|
| CAMERA_UP | O |
| CAMERA_DOWN | . (Period) |
| CAMERA_LEFT | K |
| CAMERA_RIGHT | ; (Semicolon) |
| CAMERA_UP_LEFT | I |
| CAMERA_UP_RIGHT | P |
| CAMERA_DOWN_LEFT | , (Comma) |
| CAMERA_DOWN_RIGHT | / (Slash) |
| CAMERA_TO_HERO | L |

- 키 홀드 시 연속 패닝
- 8방향 카메라 이동 지원

#### 1.8 인게임 타겟팅 시스템

- **타겟팅 커서**: Icons.TARGET 크로스헤어로 통합 표시
- **적 자동 초기화**: 스킬/아이템 사용 시 가장 가까운 적에 커서 배치
- **태그 시스템**: F(공격), G(액션), V(루팅), \`(재개)
- **이동 키로 커서 이동**: WAXD/QEZC로 게임 맵상 커서 이동
- **확정**: Enter 키로 타겟 확정

#### 1.9 컨트롤 체계 재설계

| 기능 | 키 |
|------|-----|
| 8방향 이동 | WAXD + QEZC (S = 대기/줍기) |
| UI 포커스 이동 | 화살표 키 (←↑↓→) |
| 인벤토리 탐색 | 화살표 키 (5열 그리드) |
| 퀵슬롯 | 1-6 |
| 가방 | 7-0, -, = |
| 정보/저널 | H(영웅), J(저널), R(조사) |
| 대기/휴식 | T(대기), Y(휴식) |
| 줌 | [ (축소), ] (확대) |

### 2. 데이터 동기화 (웹 데이터 공유)

URL 기반으로 게임 데이터를 다른 기기로 이동할 수 있습니다.

#### 2.1 지원 데이터

- 배지 (Badges)
- 랭킹 기록 (Rankings)
- 카탈로그/도감 (Catalog/Bestiary)
- 문서/가이드/연금술 (Documents)

#### 2.2 사용법

1. **내보내기**: DataScene -> "내보내기" 버튼 -> URL이 클립보드에 복사됨
2. **가져오기**: 복사된 URL을 브라우저 주소창에 붙여넣기 -> Enter
3. **병합 옵션**: "병합" (기존 데이터 유지) 또는 "덮어쓰기" (완전 교체)

#### 2.3 기술 사양

- 압축: pako.js (DEFLATE level 6)
- 인코딩: URL-safe Base64
- URL 길이: ~850자 (전체 데이터 포함)

### 3. 로딩 최적화

#### 3.1 지연 로드 적용

| 에셋 | 변경 전 | 변경 후 |
|------|---------|---------|
| 음악 파일 (31개) | 48MB 프리로드 | 필요 시 로드 |
| CJK 폰트 (4종) | 28MB 프리로드 | 언어별 선택 로드 |

#### 3.2 예상 로딩 시간 감소

| 사용자 | 변경 전 | 변경 후 | 절감 |
|--------|---------|---------|------|
| 영어 | ~100 MB | ~26 MB | -74% |
| 한국어 | ~100 MB | ~30 MB | -70% |
| 일본어 | ~100 MB | ~37 MB | -63% |
| 중국어 | ~100 MB | ~34 MB | -66% |

#### 3.3 언어 기반 폰트 선택

브라우저 언어 설정에 따라 필요한 CJK 폰트만 로드:
- 한국어: NotoSansKR (3.9MB)
- 일본어: NotoSansJP (11.4MB)
- 중국어 간체: NotoSansSC (7.8MB)
- 중국어 번체: NotoSansTC (5.4MB)

### 4. 다국어 폰트 시스템

#### 4.1 폰트 패밀리 우선순위

1. **Inter** - Latin, Cyrillic, Greek, Vietnamese (342KB)
2. **Noto Sans** - Latin extended (13KB)
3. **Noto Sans KR** - 한국어 (3.8MB)
4. **Noto Sans JP** - 일본어 (11MB)
5. **Noto Sans TC** - 중국어 번체 (16MB)
6. **Noto Sans SC** - 중국어 간체 (16MB)
7. **NeoDunggeunmoPro** - 픽셀 폰트 폴백 (38KB)

#### 4.2 CSS 폰트 폴백

브라우저의 CSS 폰트 폴백이 글리프별로 최적 폰트를 자동 선택합니다.
언어 설정에 따라 해당 CJK 폰트가 우선순위 상위로 이동합니다.

### 5. UI/UX 개선

#### 5.1 창 크기 표준화

| 창 유형 | 너비 |
|---------|----------|
| 정보 창 (WndMessage, WndStory 등) | 149px |
| 설정/저널/데이터 | 149px (세로) / 251px (가로) |
| WndRanking | 147px x 144px |

#### 5.2 창 활성화 스택

중첩된 창이 열릴 때 이전 창을 비활성화하고, 닫힐 때 자동으로 재활성화합니다.
- 부모 창 클릭 차단
- 탭 전환 차단 (하위 창 열림 시)

#### 5.3 창 닫기 통일 (Pattern B)

정보 표시 창(WndStory, WndJournalItem, WndBadge 등):
- ESC 키 또는 창 바깥 클릭으로 닫기
- 창 내부 클릭/Enter는 닫기 동작 없음 (실수 방지)

#### 5.4 ScrollPane 화살표 키 스크롤

스크롤 가능한 컨텐츠에서 방향키로 스크롤 가능합니다.

#### 5.5 브라우저 줌 차단

게임 내 확대/축소만 사용하도록 브라우저 줌 기능 차단:
- Ctrl+스크롤 줌 차단
- 핀치 줌 차단 (터치)
- 더블탭 줌 차단

#### 5.6 레이아웃 개선

- **조작키 설정 창**: 너비 30% 축소, 열 비율 통일
- **타이틀 화면 버튼**: 불꽃 장식과 시각적 조화 (162px 통일)
- **설정 슬라이더**: 높이 확대 (21px->28px)로 텍스트 겹침 방지
- **퀵슬롯 순번 고정**: 툴바 뒤집기와 무관하게 항상 좌->우 1,2,3,4

### 6. 텍스트 렌더링 개선

#### 6.1 하이라이트 텍스트 처리

- `_` 또는 `**`로 감싼 텍스트는 노란색으로 하이라이트
- 한국어: 하이라이트 후 띄어쓰기 유지 (서양어와 동일)
- 중국어/일본어: 하이라이트 후 띄어쓰기 생략 (CJK 특성)

#### 6.2 CJK 문자 간격 최적화

- CJK 문자별 border padding 보정
- 언어 설정과 무관하게 CJK 텍스트 일관된 렌더링

#### 6.3 줄바꿈 처리

- HIGHLIGHT_BRIDGE 토큰으로 하이라이트 경계에서 줄바꿈 기회 제공
- 서브섹션 경계 인식 네비게이션

### 7. 시드 공유 기능

#### 7.1 WndRanking 시드 표시

랭킹 기록에 던전 시드가 표시되어 동일한 맵으로 재도전할 수 있습니다.
- 일반 시드: ABC-DEF-GHI 형식
- 커스텀 시드: 사용자 지정 이름

#### 7.2 점수 분석 표시

[i] 버튼을 클릭하면 점수 구성 상세 보기:
- Progress Score
- Treasure Score
- Explore Score
- Boss/Quest Score
- 승리/도전 배수

### 8. 게임플레이 편의 기능

#### 8.1 던전 가이드 직접 표시

가이드 문서를 읽을 필요가 있을 때 저널 버튼이 깜빡이는 대신 **가이드 문서가 화면에 직접 표시**됩니다.
- 입구 저널 획득 시 -> 가이드 소개 문서 직접 표시
- 뱀(Snake) 정보 확인 시 -> 기습 공격 가이드 직접 표시
- 첫 사망 후 재시작 시 -> 죽음 가이드 직접 표시

#### 8.2 퀘스트 대화 진행 통일

NPC 퀘스트 대화창을 **ESC 키, 창 바깥 클릭, 또는 Enter** 어떤 방법으로 닫아도 대화가 "읽음" 처리되어 퀘스트가 진행됩니다.
- 이전: ESC로 닫으면 퀘스트 미진행
- 이후: 모든 닫기 방법이 동일하게 동작

#### 8.3 영웅 자동이동 중단 개선

적의 공격 시 영웅이 현재 타일까지 도착한 후 즉시 멈춥니다.
- 이전: 적 공격 후 2-3타일 추가 이동, 타일 사이에서 멈춤
- 이후: 정확한 타일 위치에서 자연스럽게 정지

#### 8.4 적 공격 애니메이션 입력 차단

적의 공격 애니메이션이 진행 중일 때 영웅 입력이 차단됩니다.
- 빠른 공격 몹(리퍼데몬, 몽크, 도둑)의 연속 공격 중 영웅이 이동하는 문제 해결
- 마우스/터치, 키보드, 컨트롤러 모든 입력 방식에 적용

#### 8.5 1층 이중 소개

새 게임 시작 시 "던전 소개" -> "하수구 소개"가 순차적으로 표시됩니다.
- 웹 버전에서 누락되던 던전 전체 소개가 정상 표시

#### 8.6 층 전환 로딩 시간 단축

로딩 화면 최소 표시 시간이 1.2초에서 1.0초로 단축되었습니다.

#### 8.7 텔레포트 자동이동 방지

텔레포트(스크롤, 함정, 비콘 등) 후 방향키 입력 시 이전 목적지로 자동이동하는 문제가 해결되었습니다.

### 9. 기타 편의 기능

#### 9.1 첫 게임 뒤로가기 네비게이션

첫 게임 시에도 영웅 선택 화면에서 ESC로 타이틀 화면 복귀 가능.

#### 9.2 모든 데이터 삭제

DataScene에서 "모든 데이터 삭제" 버튼으로 브라우저 데이터 완전 초기화.
2단계 확인 다이얼로그로 실수 방지.

#### 9.3 리플레이 기능

랭킹에서 동일한 시드, 직업, 도전 항목으로 새 게임 시작 가능.

#### 9.4 브라우저 언어 자동 감지

- `navigator.language` 기반 언어 자동 설정
- 첫 실행부터 브라우저 언어로 표시

#### 9.5 시스템 폰트 선택

- 픽셀 폰트(NeoDunggeunmoPro) 또는 시스템 폰트 선택 가능
- 시스템 폰트: system-ui, -apple-system, Segoe UI, Roboto 등

#### 9.6 가져오기 후 자동 새로고침

데이터 가져오기 완료 시 페이지 자동 새로고침으로 게임 데이터 즉시 반영.

#### 9.7 해시 프래그먼트 호환성

- URL hash를 sessionStorage에 저장 후 URL에서 제거
- 게임 실행 중 import URL 붙여넣기 지원
