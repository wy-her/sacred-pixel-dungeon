# 036. 광전사 분노 메커니즘 개편

**날짜**: 2026-04-02

## 개요

광전사(Berserker)의 광포화(Berserk) 발동 조건과 쿨다운 메커니즘을 대폭 수정하여 전략적 유연성을 높였습니다.

---

## 변경 사항

### 1. 발동 조건 변경
- **기존**: 분노 100%에서만 광포화 발동 가능
- **변경**: 분노 **10% 이상**에서 광포화 발동 가능

### 2. 쿨다운 메커니즘 변경
- **기존**: 광포화 후 항상 100턴 고정 쿨다운
- **변경**: **소모한 분노% = 쿨다운 턴** (예: 50% 분노로 발동 시 50턴 쿨다운)

### 3. ENDLESS_RAGE 특성 변경
- **기존**: 100% 초과 분노당 방어막 +1%, 쿨다운 -1%, 데미지 캡 +50% 유지
- **변경**:
  - 쿨다운 감소 효과 **제거** (분노 기반 쿨다운으로 대체)
  - **데미지 캡 제거** (150% 분노 = +75% 데미지)
  - 방어막 보너스 유지 (150% 분노 = 1.5배 방어막)

---

| `messages/actors/actors.properties` | angered_desc, berserker_short_desc, berserker_desc, endless_rage.desc 업데이트 |

---

## Balance Impact

### 장점
| 상황 | 효과 |
|------|------|
| 긴급 상황 | 낮은 분노로도 방어막 획득 가능 (약하지만 빠른 쿨다운) |
| 전략적 선택 | 리스크/리워드 트레이드오프 명확화 |
| 초반 생존력 | 100% 분노 전에도 위기 대응 가능 |
| ENDLESS_RAGE | 더 높은 데미지 보너스 (+75% at 150%) |

### 트레이드오프
| 분노 | 데미지 보너스 | 방어막 | 쿨다운 |
|------|--------------|--------|--------|
| 10% | +5% | 약함 | 10턴 |
| 50% | +25% | 중간 | 50턴 |
| 100% | +50% | 강함 | 100턴 |
| 150% (ENDLESS_RAGE) | +75% | 1.5배 | 150턴 |

## Test Scenarios

1. **분노 10% 발동**: 액션 버튼 표시, 10턴 쿨다운 확인
2. **분노 50% 발동**: 50턴 쿨다운, +25% 데미지 확인
3. **분노 100% 발동**: 100턴 쿨다운, +50% 데미지 확인
4. **분노 150% (ENDLESS_RAGE)**: 150턴 쿨다운, +75% 데미지, 1.5배 방어막 확인
5. **분노 9% 미만**: 액션 버튼 미표시 확인
